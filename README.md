# RTSPBruter 🔪 

RTSPBruter is python tool used to bruteforce authentification of Real Time Streaming Protocol (RTSP).


# Installation 🏗 

pip3 install -r requirements.txt


# Usage 🛠 

python3 -u usernames.txt -P passwords.txt -i 127.0.0.1 -p 554
